Creando una lista ordenada

  1. Establecer la etiqueta "ol"
  2. Poner la etiqueta "li" a cada paso
  3. Escribir los pasos y cerrar la lista

Creando una lista no ordenada

  • Escribir los pasos y cerrar la lista
  • Poncer la etiqueta "li" a cada elemento
  • EStablecer la etiqueta "ul"
  • En estas listas el orden de los factores no altera el producto
